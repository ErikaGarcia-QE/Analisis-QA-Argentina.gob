# Checklist Responsive

Este checklist permite validar que el sitio funcione correctamente en diferentes dispositivos.

## ✔ Mobile (≈ 360px)

- [ ] No hay desbordes horizontales  
- [ ] El menú hamburguesa funciona correctamente  
- [ ] Las imágenes se adaptan sin deformarse  
- [ ] Los textos no quedan demasiado pequeños  
- [ ] Los botones son fáciles de tocar  

## ✔ Tablet (≈ 768px)

- [ ] El layout se reorganiza correctamente  
- [ ] Las columnas se adaptan sin romperse  
- [ ] Los elementos mantienen proporciones adecuadas  

## ✔ Desktop (≥ 1366px)

- [ ] El contenido se distribuye correctamente  
- [ ] No hay espacios vacíos excesivos  
- [ ] Las tarjetas y secciones se alinean bien  

## ✔ General

- [ ] Los elementos mantienen coherencia entre dispositivos  
- [ ] Las imágenes son fluidas (max-width: 100%)  
- [ ] Los formularios se adaptan al ancho disponible  
- [ ] No hay elementos superpuestos
