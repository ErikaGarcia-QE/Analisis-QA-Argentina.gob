# Checklist de UI (Interfaz de Usuario)

Este checklist permite validar la calidad visual y la coherencia de la interfaz del sitio.

## ✔ Elementos visuales

- [ ] Los textos son legibles y con tamaño adecuado  
- [ ] Las tipografías son consistentes en todo el sitio  
- [ ] Los colores respetan la identidad visual  
- [ ] No hay textos cortados o superpuestos  
- [ ] Las imágenes se ven nítidas y sin pixelación  
- [ ] Los íconos son claros y coherentes  
- [ ] Los botones tienen suficiente padding  

## ✔ Navegación y estructura

- [ ] El menú es visible y fácil de usar  
- [ ] Los enlaces están correctamente identificados  
- [ ] El footer contiene información relevante  
- [ ] No hay elementos desalineados  
- [ ] No hay espacios vacíos excesivos  

## ✔ Interacción

- [ ] Los botones tienen estados hover/active  
- [ ] Los formularios muestran mensajes de error claros  
- [ ] Los enlaces cambian de estilo al pasar el mouse  
- [ ] Los elementos clickeables son distinguibles  

## ✔ Contenido

- [ ] No hay errores ortográficos  
- [ ] Los textos son claros y concisos  
- [ ] Las imágenes tienen relación con el contenido
