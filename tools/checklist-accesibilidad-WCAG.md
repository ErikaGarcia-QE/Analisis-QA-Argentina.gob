# Checklist de Accesibilidad – Basado en WCAG 2.1 AA

Este checklist ayuda a validar que el sitio sea accesible para todas las personas.

## ✔ Perceptible

- [ ] Todas las imágenes informativas tienen texto alternativo (alt)  
- [ ] El contraste entre texto y fondo cumple 4.5:1  
- [ ] Los textos no dependen solo del color para transmitir información  
- [ ] Los videos tienen subtítulos (si aplica)  

## ✔ Operable

- [ ] El sitio se puede navegar completamente con teclado  
- [ ] El foco es visible en todos los elementos interactivos  
- [ ] No hay contenido que parpadee rápidamente  
- [ ] Los menús desplegables funcionan sin mouse  

## ✔ Comprensible

- [ ] Los formularios tienen etiquetas claras  
- [ ] Los mensajes de error son descriptivos  
- [ ] El orden de lectura es lógico para lectores de pantalla  
- [ ] Los enlaces tienen nombres descriptivos (no “clic aquí”)  

## ✔ Robusto

- [ ] El HTML tiene estructura semántica correcta  
- [ ] Los encabezados siguen un orden jerárquico (H1 → H2 → H3)  
- [ ] Los roles ARIA están correctamente aplicados
