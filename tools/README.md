# 🧰 Herramientas y Checklists de QA

Esta carpeta contiene recursos utilizados durante el análisis QA, incluyendo listas de verificación y guías prácticas.

## 📌 Contenido

- `checklist-UI.md`  
  Validaciones visuales, consistencia, alineación, tipografía, colores.

- `checklist-accesibilidad-WCAG.md`  
  Criterios básicos de accesibilidad (WCAG 2.1 AA).

- `checklist-responsive.md`  
  Validaciones en diferentes tamaños de pantalla.

## 🎯 Objetivo

Proveer herramientas que complementen el análisis y permitan evaluar:

- Calidad visual  
- Accesibilidad  
- Adaptabilidad  
- Buenas prácticas de diseño y contenido
