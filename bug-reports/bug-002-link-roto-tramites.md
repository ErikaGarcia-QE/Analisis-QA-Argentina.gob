# BUG-002 – Enlace roto en sección Trámites

**Severidad:** Media  
**Prioridad:** Media  

## Descripción
Al intentar acceder al enlace “Iniciar trámite” en ciertos servicios, el sitio devuelve un error 404.

## Pasos para reproducir
1. Ir a https://www.argentina.gob.ar/tramites  
2. Seleccionar un trámite al azar  
3. Hacer clic en “Iniciar trámite”  

## Resultado esperado
El enlace abre el formulario o página correspondiente al trámite.

## Resultado obtenido
Se muestra un error 404 o página no encontrada.
